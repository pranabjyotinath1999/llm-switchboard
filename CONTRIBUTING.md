# Contributing to llm-switchboard

First off, thank you for considering contributing to llm-switchboard! It's people like you that make it a great tool for the AI community.

## 🚀 How to Contribute

### Reporting Bugs
- Check the [issues](https://github.com/Uo1428/llm-switchboard/issues) to see if the bug has already been reported.
- If not, open a new issue with a clear title and description, including steps to reproduce.

### Suggesting Enhancements
- Open a new issue to discuss the enhancement before starting work.
- Provide a clear rationale for the change.

### Pull Requests
1. Fork the repository.
2. Create a new branch (`git checkout -b feature/amazing-feature`).
3. Make your changes.
4. Ensure tests pass (`bun run test`).
5. Commit your changes (`git commit -m 'Add some amazing feature'`).
6. Push to the branch (`git push origin feature/amazing-feature`).
7. Open a Pull Request.

## 🛠️ Development Setup

```bash
# Clone the repo
git clone https://github.com/Uo1428/llm-switchboard.git
cd llm-switchboard

# Install dependencies
bun install

# Run tests
bun run test
```
